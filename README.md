# Installation
```
pip3 install -r requirements.txt
```

# Usage
```
python3 main.py [NETAPIX_OUTPUT_PATH] [LABELS_PATH]
```
| Param | Comment |
| ------------- | ------------- |
| NETAPIX_OUTPUT_PATH | path to directory with netapix outputs (\*.npo)|
| LABELS_PATH | path to directory with target values (\*.txt or \*.jpg)|
